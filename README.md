## Results Analytics SPA

A single-page application built with **React + Vite** that lets students look up their JNTUH academic results by roll number.

### Features

- Search by roll number and view student info, semester-wise subject marks, grades, credits, and SGPA
- Dark-themed, responsive UI built with Tailwind CSS
- Real-time data from the JNTUH results API

### Getting Started

```bash
cd results-analytics
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
